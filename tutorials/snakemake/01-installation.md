
# Tutorial: Getting Started with Your Tool

This tutorial is meant to be run on the ssh server, please make sure to run the following command first to set up your environment:

On the ssh server, run:
```bash
conda-shared
conda activate cogpy
```

test run
```bash
snakemake -h
```
This should display the help message for Snakemake, confirming that it is installed correctly.


---

For local installations, follow the instructions in the [Snakemake installation guide](https://snakemake.readthedocs.io/en/stable/tutorial/setup.html).
